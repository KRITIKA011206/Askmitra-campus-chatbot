# Askmitra-campus-chatbot
AI-powered multilingual campus chatbot for Banasthali Vidyapith built using HTML, CSS, JavaScript and Chatbase

The project focuses on creating a **clean, responsive frontend** and integrating an AI chatbot using **Chatbase iframe embedding**, without using a custom backend.

---

 Key Features
- 24/7 AI-based campus assistance
- Supports multiple languages (English, Hindi, and local languages)
- Floating chatbot button for easy access
- Clean and minimal user interface
- Fully responsive on mobile and desktop
- Can be easily embedded into any college website

---

 Tech Stack
- HTML5
- CSS3
- JavaScript
- Chatbase (iframe-based integration)
---

 How It Works
- Users open the website and click the floating **AskMitra** chat button.
- JavaScript controls the chatbot toggle and UI interactions.
- The chatbot appears as an embedded Chatbase iframe and responds in real time.
- The entire application runs on the frontend with no backend setup.
